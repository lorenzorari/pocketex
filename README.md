# Built with

- [React](https://fr.reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [SASS](https://sass-lang.com/)
- CSS modules
- [PokéAPI](https://pokeapi.co/)
- [Netlify](https://www.netlify.com/)

# Installing

1.  Install dependencies

        yarn install

2.  Run the app locally

		yarn start
